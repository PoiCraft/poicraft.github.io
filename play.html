# 加入白名单

  1. 加入官方 QQ群(769885907 (opens new window))
  2. 将 群名片 修改为 Xbox ID
  3. 向群内管理员申请白名单

# 在主机上加入本服务器

# 概述

由于主机平台的安全性策略，一般来讲，你是不能加入除“精选服务器”外的外部服务器的。但是你有如下方法：通过DNS劫持覆盖精选服务器 (opens new window)或者模拟局域网联机 (opens new window)来让主机加入此服务器
